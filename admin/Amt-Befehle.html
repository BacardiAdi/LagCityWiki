Amt Befehle 

 

Hier findest du die aktuellen Befehle für das Amt.

  • Beruf verlernen: Stream des Spielers an → /cp → permissions → Spieler wählen → RPSkills → unlern (rechte Spalte) →  Konfuzius verlernen → /cp → unlern wieder deaktivieren

 

Befehle für das Stellen oder bearbeiten von NPC´s

/npc_add => erstelle einen neuen NPC und bearbeite ihn

/npc_edit [id] => bearbeite den NPC, den du gerade ansiehst, oder gib die ID an

/npc_remove [id] => lösche den NPC, den du gerade ansiehst, oder die angegebene ID

/npc_sound => Ton aufnehmen, damit dein NPC es sagt

/npc_end => beende die Bearbeitung eines NPCs

/npc OPTION VALUE => setze den Wert einer Option deines NPCs

/npc_reset => entfernt alle NPCs

/npc_pathtest => NPC-Pfad folgen

/npc_list => listet alle NPCs auf

/npc_way [id] => zeichnet den Pfad des NPCs, den du gerade ansiehst, oder die angegebene ID
