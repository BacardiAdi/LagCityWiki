APU Commands

/createclerk - Creates a clerk at your current location 

/removeclerk - Removes a clerk that you are looking at. 

/getclerkid - Ability to get the id of the clerk in the data file, so you can edit rotation or position if needed.

 

 

NPC Commands

/npc_list - Liste der NPCs

/npc_edit “ID” - Starten des NPC “Editors”

/npc spawn “here” - Spawnt den NPC auf deiner Position

/npc_end - Beendet das NPC editieren
 

 

Copy/Paste Plugin

/copy + Name - Kopieren des Gebäudes, welches du gerade anschaust

/copylist - Zeigt dir die Liste aller Kopierten Gebäude auf dem Server

/paste + Name - Einfügen des Gebäudes

/paste + Name + height z.B. 1 - Einfügen des Gebäudes + justieren der Höhe um 1

/pasteback - Einfügen eines Gebäudes an der vorherigen Stelle (Stelle des 1. Kopierens)
