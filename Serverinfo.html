So trittst du dem Server bei - Authentifzierungsleitfaden

 

Checkliste:

  • Rust (über Steam bereits Installiert)
  • Discord (Account + PoT-Discord beigetreten)

 

3-Schritte Plan:

  • 1. Schritt
  • 2. Schritt
  • 3. Schritt

 

Schritt 1: Rust starten & Server beitreten

 

Möglichkeit A:  Über die Serverliste

  1. Starte Rust
  2. Klicke im Hauptmenü auf Play
  3. Öffne Servers
  4. Suche nach: LagCity RP [GER] by PoT
  5. Klicke auf Join/Beitreten

 

Möglichkeit B: Direkt über die Konsole

  1. Starte Rust
  2. Drücke F1 (die Konsole wird geöffnet)
  3. Gib folgenden Befehl ein und drücke Enter: client.connect 88.198.7.190:28015

 

Schritt 2: Prüfen, ob der Server offen ist (WICHTIG)

 

Du kannst nur spielen, wenn der Server geöffnet ist!

  1. Öffne den PoT-Discord
  2. Schaue rechts in die Mitgliederliste
  3. Achte auf den Bot Whitelist

 

Folgende Bezeichnungen sind relevant:

Server offen

→ Du kannst beitreten

Whitelist aktiv

→ Die Server Whitelist ist aktuell geschlossen

 

Was ist die Whitelist?

Die Whitelist ist eine Freigabe.
Nur wenn sie aktiv ist, kannst du den Server betreten.
Sie wird von einem Admin geöffnet.

Admins sind meistens abends online!
Außerhalb dieser Zeiten kannst du ein Ticket im Discord erstellen.

 

Schritt 3: Authentifizieren (Pflicht)

So geht's

  1. Betrete den Server
  2. Öffne den Chat
  3. Schreibe: /auth
  4. Du bekommst einen Code
  5. Code kopieren
  6. Öffne Discord 
  7. Senden den Code als Direktnachricht (PM) an den RustAuth-Bot

→ Und Ferttiiiggg! Viel Spaß beim spielen

 

 

 

 

 

 

Falls du dem Bot keine Nachricht senden kannst

  1. Öffne Discord → Benutzereinstellungen (Zahnrad-Symbol ⚙️)
  2. Privatsphäre & Sicherheit
  3. Aktiviere folgendes: Direktnachrichten von Servermitgliedern erlauben
