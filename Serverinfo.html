So trittst du dem Server bei:

 

 

  1. Server-Suche:
    • Starte Rust und suche im Server-Menü nach dem Namen „LagCity RP [GER] by PoT“.
  2. Direktverbindung über die Konsole:
    • Drücke im Hauptmenü von Rust die Taste F1, um die Konsole zu öffnen.

Gib den Befehl client.connect 142.132.194.156:28015 in die Konsole ein und drücke Enter.

 

 

 

 

 

Wichtige Schritte:

 

1. 

Um Zugang zum Server zu erhalten, ist es wichtig zu wissen, dass du nur dann beitreten kannst, wenn ein Admin anwesend und der Server geöffnet ist. Diesen Zustand kannst du ganz einfach überprüfen, indem du einen Blick in den Discord wirfst. Auf der rechten Seite in Discord findest du die Mitgliederliste, die dir zeigt

Damit du sehen kannst, ob ein Admin online ist und der Server geöffnet wurde, musst du sicherstellen, dass die Mitgliederliste in deinem Discord sichtbar ist. Sollte diese nicht direkt angezeigt werden, kannst du sie  einfach einblenden:

 

Bevor deine Abenteuer auf der Insel beginnen können, ist es wichtig, dass du dich zuerst mit deinem Discord-Konto im PoT- Discord authentifizierst. Dieser Schritt stellt sicher, dass du Zugang zum Server erhältst, selbst wenn eine Whitelist aktiviert ist.

Um den Authentifizierungsprozess zu starten,gebe den Befehl /auth direkt in den Chat ein. Nachdem du diesen Befehl eingegeben hast, wird dir ein  Buchstabencode zugewiesen. Diesen Code musst du dann kopieren und als direkte Nachricht an den RustAuth-Bot innerhalb des Discord-Servers senden:

Durch das Senden dieses Codes an den Bot wird dein Discord-Konto mit deinem Spielerprofil auf dem Server verknüpft. Diese Verknüpfung ist essentiell, um sicherzustellen, dass nur authentifizierte Spieler Zugang zum Server erhalten, und hilft dabei, eine positive und faire Spielumgebung für alle zu gewährleisten.

Folge diesen Schritten sorgfältig, um sicherzustellen, dass du ohne Probleme auf den Server zugreifen und das Spiel in vollen Zügen genießen kannst. Solltest du auf irgendwelche Schwierigkeiten stoßen oder weitere Fragen haben, zögere nicht, das Support-Team auf dem Discord-Server um Hilfe zu bitten. Wir sind hier, um dir ein erstklassiges Spielerlebnis zu bieten!

 

Bist Zuerst musst du dich mit Discord Discord-Server. authentifizieren, indem du den Befehl /auth im Rust-Chat eingibst. Den Buchstabencode, den du daraufhin erhältst, sendest du als Nachricht im Discord an den RustAuth-Bot, welcher in der Mitglieder-/Memberliste ganz oben zu finden ist.

Um dem Server joinen zu können, muss zuerst ein Administrator auf der Insel sein und die Whitelist deaktivieren. Erst wenn 

diese Voraussetzungen erfüllt sind, ist das Joinen auf dem Server möglich.
