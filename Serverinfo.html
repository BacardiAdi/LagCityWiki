So trittst du dem Server bei:

 

 

  1. Server-Suche:
    • Starte Rust und suche im Server-Menü nach dem Namen „LagCity RP [GER] by PoT“.
  2. Direktverbindung über die Konsole:
    • Drücke im Hauptmenü von Rust die Taste F1, um die Konsole zu öffnen.

Gib den Befehl client.connect 142.132.194.156:28015 in die Konsole ein und drücke Enter.

 

 

 

 

 

Wichtige Schritte:

 

  • Um Zugang zum Server zu erhalten, ist es wichtig zu wissen, dass du nur dann beitreten kannst, wenn ein Admin anwesend und der Server geöffnet ist. Das kannst du ganz einfach überprüfen, indem du einen Blick in den PoT- Discord wirfst. Auf der rechten Seite im Discord findest du die Mitgliederliste -  die Mitgliederliste in deinem Discord muss sichtbar sein. Sollte diese nicht direkt angezeigt werden, kannst du sie  einfach einblenden:

 

 

Hier kannst du sehen, ob der Server offen ist: 

  • Bevor deine Abenteuer auf der Insel beginnen können, ist es wichtig, dass du dich zuerst  im PoT- Discord authentifizierst. Dieser Schritt stellt sicher, dass du Zugang zum Server erhältst, selbst wenn die Whitelist aktiviert ist.

 

 

 

  • Um den Authentifizierungsprozess zu starten, gebe den Befehl /auth direkt im Spiel den Chat ein.   Nachdem du diesen Befehl eingegeben hast, wird dir ein  Buchstabencode zugewiesen. Diesen Code   musst du dann kopieren und als direkte Nachricht an den RustAuth-Bot innerhalb des Discord-Servers   senden: 

 

 

 

 

Falls du dem RustAuthBot keine Nachricht schicken kannst, musst du unter deinen Benutzereinstellungen    ⇢

 

 

 

 Diese Verknüpfung ist essentiell, um sicherzustellen, dass nur authentifizierte Spieler Zugang zum Server erhalten.

 

Solltest du auf irgendwelche Schwierigkeiten stoßen oder weitere Fragen haben, zögere nicht, ein Ticket zu öffnen.
